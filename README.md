| 文件夹       | 说明                    | 描述                        |
| ------------ | ----------------------- | --------------------------- |
| `config`     | 配置包                  | config.yaml对应的配置结构体 |
| `controller`    | controller层               | 业务逻辑 |
| `dto`    | dto层               | 控制数据返回字段 |
| `middleware` | 中间件层 | 用于存放 `gin` 中间件代码 |
| `model`      | 模型层                  | 模型对应数据表              |
| `repository`      | 交互层                  | 与数据库进行交互              |
| `response`      | 响应处理                  | 请求响应封装              |
| `utils`      | 工具包                  | 工具函数封装            |
| `vo` | vo层 | 请求参数绑定验证 |
